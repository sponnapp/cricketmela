## 🎯 FEATURE OVERVIEW

### What It Does
Automatically charges newly assigned users 10 points per unvoted completed match, and distributes those points to winners using the 1:1 ratio.

### When It Triggers
1. User approval with season assignment
2. Direct season assignment to existing user

### Key Metrics
- **Auto-loss amount**: 10 points per match
- **Distribution**: Proportional to winner stakes
- **Processing**: Non-blocking, async
- **Error handling**: Graceful, logged
